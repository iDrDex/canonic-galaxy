# Welcome to your canonic-galaxy

This repo is your personal governance graph — seeded by LAUDE, grown by your conversations.

## What is this?

Every time you chat with [LAUDE](https://laude.canonic.org), the session closes with a PR here. Axioms you've captured land in `INTEL/`. Transcripts land in `TRANSCRIPTS/`. When you're ready to make something permanent, you edit `CANON.md`.

## Structure

```
CANON.md          — your personal axiom set (edit this)
TRANSCRIPTS/      — session transcripts, one file per chat
INTEL/            — extracted axioms accumulator, one file per month
magic-manifest.json — your federation identity (do not edit manually)
```

## Federation

Your galaxy is yours. To share a scope with another LAUDE user, ask the LAUDE agent to `share_scope`. To let others discover your galaxy, set `federation: open` in your manifest via the LAUDE settings.
